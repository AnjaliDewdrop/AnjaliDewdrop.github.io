H2OnTheWay

System that detects water levels in pipes upstream from communal reservoirs in developing regions.

Built autonomous embedded system to detect water levels using Python-Arduino Serial Bridge.
Automated system to send an SMS to locals that shows their location relative to active reservoirs.
Received 1st place, and the Twilio Innovation Prize, out of over 500+ participants.

  • Date: February 2019

VisionSense

Wearable technology that tactically guides users with visual impairment.

Developed IoT outfit that senses obstructions using Ultrasonic sensors and communicates distance and direction of obstructions through vibrators in the outfit via Raspberry Pi

  • Date: March 2019

Digi-Syringe

Smart syringe that tracks total liquid dispensed for medication preparation

Used Arduino and Potentiometer alongise 3D printed Components to create casing for a syringe that detects how much solution has been dispensed over time.
Achieved a 99% accuracy on average for all viscocities of liquid solutions attempted.

  • Date: September to December 2018

Safe and Sound

AR based virtual assistant aiding users with Parkinson’s Dementia manage symptoms.

Built Unity app for Bose Frames to detect falls, tremors and custom gestures and provide audio support
Programmed AR feature in Unity to gamify process of locating medication and taking correct dose

  • Date: November 2019

ECG Peak Detector

Used AI to detect features of ECG Signal

Built pattern recognition algorithm using Python Jupyter Notebook and SciKit to analyze data corresponding to an ECG signal and detect various features like irrecular rates, PQRS peaks, etc.

  • Date: October 2018

Phonomyography POC

Analogue circuit design controlled by phonomyographic singals (audio frequencies generated by human muscles)

Designed an analogue circuit that detects audio frequency generated by human muscles, filters them for relevant contractions, and rotates motors based on the speed of contraction
Used an oscilloscope to test for accuracy of RC filters involved

  • Date: July 2019

Glucose Run

Web Platformer Game to put a spotlight on symptoms of Type 1 Diabetes in children

Built game in JS using Phaser Library.
Responsible for power-up insulin respawning features and generating realistic function for tracking glucose concentration of primary player

  • Date: May 2018

Cochlear Audio Filter

Software mimic functionality of a cochlear implant used to detect and transmit audio to user's nervous system

Built various types of signal filters in MATLAB and ranked them based on noise cancellation, detection of human voice, and clarity of pitch and syllables.

  • Date: May to August 2019

Kinnekt

ML based Web tool that teaches children circuitry basics through virtual circuit building activities

Built Node.js backend that takes snapshots of completed circuitry from user and compares them for validity using Microsoft Custom Vision ML Image Recognition model with an 85% accuracy rate

  • Date: March 2019

Clinical Connections

Web Tool to increase process efficiency for clinics in Canada sending supplies to under-supplied clinics in other regions

Designed UI and built functional web tool to track supplies and allow universal clear communication between all stakeholders involved in transaction of supplies.
Used HTML and CSS, Node.JS and SQL to build tool.
Won 3rd place out of 300 participants.

  • Date: September 2018

StepItUp

Mechanical tool to reduce risk of child athletes developing Little League Elbow disorder

Used SolidWorks to design prototypes after conducting user interviews and extensive research.
Built final design using raw materials in the machine shop.
Achieved a 95% improvement in throwing angle responsible for causing this disorder in athletes.

  • Date: September to December 2017
